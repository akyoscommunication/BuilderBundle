# BuilderBundle

Page builder visuel pour Symfony : composants réutilisables, templates, valeurs traduisibles et rendu front via Twig. Dépend de CoreBundle.

## Compatibilité

- PHP >= 8.4
- Symfony ^8.0

## Dépendances Akyos

- `akyos/core-bundle` ^8.0

## Installation

```bash
composer require akyos/builder-bundle:^8.0
```
